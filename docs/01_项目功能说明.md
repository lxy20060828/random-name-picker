# 随机点名器 功能说明文档

## 1. 项目概述

- 项目名称：随机点名器
- 作者姓名/学号：[姓名] / [学号]
- 开发周期：2026 年 6 月 - 2026 年 7 月
- 项目简介（200字以内）：本项目是一个面向课堂互动场景的随机点名工具。教师可以导入不少于 40 人的学生名单，进行随机抽取、查看点名历史和统计数据。项目使用 Vue 3、TypeScript、Composition API、Vite 和 Element Plus 开发，数据通过 localStorage 持久化，并提供每日一句免费外部 API、批量导入、数据统计和名单导出等扩展功能。

## 2. 需求分析

### 2.1 用户角色

- 教师：导入学生名单、开始随机点名、查看点名记录、导出名单。
- 学生：作为被点名对象，参与课堂互动。

### 2.2 核心功能需求

- 学生名单管理：添加、编辑、删除、搜索、清空、导出。
- 批量导入：支持 Excel、CSV、TXT，自动识别姓名列，跳过重复和无效数据。
- 随机点名：点击按钮随机抽取学生，显示动画和最终结果。
- 点名记录：记录学生姓名和点名时间，保留最近 100 条记录。
- 数据统计：展示学生总数、点名次数、已点人数、人均次数和高频点名学生。
- 数据持久化：使用 localStorage 保存学生名单和历史记录。

### 2.3 扩展功能需求

- 记录管理：支持删除单条点名记录，也可以一键清空全部记录。
- 外部 API：接入 Hitokoto 一言免费接口，展示课堂鼓励语。
- 响应式布局：适配手机、平板和电脑。
- UI 组件库：使用 Element Plus。
- 单元测试：使用 Vitest 测试导入解析、去重和随机抽取函数。

## 3. 功能模块设计

| 模块 | 功能 | 输入 | 输出 | 说明 |
|:---|:---|:---|:---|:---|
| 名单管理 | 添加、编辑、删除、搜索学生 | 姓名、学号、班级、备注 | 学生列表 | 支持表单校验和重复提示 |
| 批量导入 | 解析 Excel/CSV/TXT | 文件或粘贴文本 | 可导入名单、错误提示 | 自动识别表头和姓名列 |
| 随机点名 | 随机抽取学生 | 学生数组、点击事件 | 被点名学生 | 抽取后累计 pickCount |
| 点名记录 | 保存抽取历史 | 被点名学生 | 历史列表 | 最近记录排在前面 |
| 数据统计 | 统计点名情况 | 学生列表、历史记录 | 统计卡片 | 使用 computed 计算 |
| 记录管理 | 删除单条记录、清空记录 | 记录 id、点击事件 | 更新后的历史列表 | 删除单条记录时同步修正点名次数 |
| 每日一句 | 调用免费外部 API | 无 | 鼓励语 | 失败时使用本地备用语句 |

## 5. 非功能需求

- 性能：页面加载目标小于 2 秒，主要数据存储在浏览器本地。
- 兼容性：支持 Chrome、Firefox、Edge 最新版。
- 可用性：导入、添加、点名、清空等操作均有反馈；导入错误会显示提示。
